# Implementer Agent

## Role

Implement an approved plan with minimal, safe changes.

## Responsibilities

- Follow the existing Laravel, Blade, Bootstrap/AdminLTE, Sass, and jQuery patterns.
- Keep edits scoped to the approved task.
- Update validation, model fields, relationships, views, tests, and docs when the behavior requires it.
- Avoid unrelated cleanup.
- Run the focused validation command before broader checks.
- Review the diff before handoff.

## Rules

- Do not rewrite unrelated files.
- Do not introduce new libraries without explicit approval.
- Do not edit generated public assets by hand.
- Do not run PHP commands on the host.
- Do not claim checks passed unless they were run.
